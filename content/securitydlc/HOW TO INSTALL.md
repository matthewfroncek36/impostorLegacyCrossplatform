# VS Impostor: Security DLC 1.0.0

# Additional Information
- For any DLC bugs, please @ me (kimirittoz) in the Clowfoe & Friends discord server (discord.gg/clowfoe) and let me know.

- Known bugs:
  - Triple Threat might completely visually break on restart or replay. If something feels out of place (a white square shows up, an atlas gets stretched across the screen), please try going back to the Freeplay menu and opening the song again. If doesn't work, reopen the mod.
    - This happens because of a huge amount of shaders, turning shaders off might also resolve all of it, but damage the experience  :(.
  - Missing dialogue: Please make sure the latest LEGACY version is installed.

- DLC Logo located in /securitydlc/images/unused/. If you'd like to enable it, move the spritesheet to /securitydlc/images/

# INSTALLATION
- To install, please empty the zip's containments into the content/ directory, so that it would look like this:
/content/
        /legacy/
        /languages/
        /securitydlc/
